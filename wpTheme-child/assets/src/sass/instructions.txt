1. Install sass - npm install node-sass --save-dev
2. Create the package.json
3. Add to the Package.json:
          "scripts": {
               "compile:sass": "node-sass wpTheme-child/assets/src/sass/main.scss wpTheme-child/assets/dist/css/style.css -w"
          },
4. Execute:
     cd C:\xampp\htdocs\wordpress\wp-content\themes\wpTheme
     npm run compile:sass

optional:
1. npm install bootstrap
